# Habit Tracker

## Description
The Habit Tracker project aims to provide an intuitive and efficient digital solution to help users monitor, manage, and improve their daily routines.

## Features
- Habit logging
- Streak tracking
- Calendar view
- Reminders
- Bad habit timer
- Visual analytics

## Technologies Used
- Backend: Node.js, Express, MongoDB
- Frontend: React
